Overview

This procedural animation system is my 2nd iteration. It is a work in progress at the moment. However, the first iteration can be found on my site here!

Procedural animation is a great fascination to me as it is a great way to quickly add personality to a character just by tweaking some settings. In this iteration, I have improved the handling and responsiveness of the leg animation. In addition, this system works on any surface for example: ceilings and walls.
