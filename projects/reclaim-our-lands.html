Overview

'Reclaim Our Lands' is my 1st year masters project made on our own with the goal of using all the game development disciplines.

This is to make us more 'T-Shaped' and therefore specialised in atleast one practice but able to step into other disciplines if required.

My Involvement

Every interactable in this project derives from a single 'base class' called Entity. From that entity class, we have structures, characters, resources and more! This system is paired with a variety of C# interfaces to help improve code modularity and readability. This approach allowed me to easily create tooltip systems and UI which revolve around this system; providing information to the player.

The various characters in this game make use of steering behaviours to navigate around the world and avoid obstacles. In addition, their class is programmed to take in instructions. This is especially prevalent in the 'Human' characters because the player can give them alot of different instructions.

The dialogue system uses a profile system to change the speakers pitch and speaking speed to give each speaker its own personality.

The tutorial system is built ontop of the dialogue system and waits for various actions to be completed until the next dialogue continues.
