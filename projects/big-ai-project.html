Overview

'Big AI project' is my 3rd year project made on our own for a module (Gam340). The source-code can be found on Github here!

The idea behind this module was to make a strong artefact project to add to our portfolios. For my project I wanted to look into adding sensory based AI using sight and hearing sensors and making those work with a behaviour tree decision algorithm to perform various behaviours.

In addition, steering behaviours were added to the characters in order to try avoid them walking into obstacles.
